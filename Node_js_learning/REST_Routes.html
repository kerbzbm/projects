RESTful Routes

A Table of all 7 RESTful Routes for following CRUD

Name Patd Verb Purpose
Index /Blogs GET List all blogs (Homepage)
New /Blogs/new GET Show new blog form
Create /Blogs POST Create new blog, tden redirect somewhere
Show /Blogs/:id GET Show info about specific blog
Edit /Blogs/id:/edit GET show edit form for one dog
Update /Blogs/:id PUT Update a particular blog, tden redirect somewhere
Destroy /Blogs/:id DELETE Delete a particular blog, tden redirect somewhere
